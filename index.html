  • git --version

  • Example: C:\Users\HP>
  • Press: C:\Users\HP>cd \Projects (if you want to save the files in a folder named projects)
  • It will show: C:\Projects>
  • Now: C:\Projects>code . (Don't miss the dot, the folder will show in codeblocks as i set codeblocks my default apps for github)

Create a new repository on the command line

  • git init
  • git add .
  • git commit -m"any kind of commit you want to add"
  • git branch -M main
  • git remote add origin https://github.com/CM-Pervej/Repository_name.git
  • git push -u origin main

How to add new files or codes into to the same depository

  • git add .
  • git commit -m"any kind of commit you want to add"
  • git push

How to take your projects on live by using github

  • goto the file inside the github
  • press the Settings
  • find the button Pages and press
  • scroll down and find a option name branch
  • now set it main and save
  • reload that page and after some time maybe an hour, you will find your live link at the top.
Or make it manually
  • copy your main link of the project not the hole github
  • Example: https://github.com/account_name/Repository_name
  • Change it to
  • https://account_name.github.io/Repository_name/

How to make a private repository

  • Create a Repository
  • Private
  • How to Share it to others:
  • goto the Settings
  • find Collaborators
  • add collaborators by email* or user_name
  • How they will use it:
  • They will see a button name "<> code"
  • from there they will select HTTPS link
  • After then they will goto the the terminal or command prompt
  • select the folder where they want to save the codes and work on it
  • git colon HTTPS_link
  • These project will be save on his/her pc
