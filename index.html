First check if the git is enable

  • git --version

Create a new repository on the command line

  • git init
  • git add .
  • git commit -m"any kind of commit you want to add"
  • git branch -M main
  • git remote add origin https://github.com/CM-Pervej/Repository_name.git
  • git push -u origin main

How to add new files or codes into to the same depository

  • git add .
  • git commit -m"any kind of commit you want to add"
  • git push

How to take your projects on live by using github

  • goto the file inside the github
  • press the Settings
  • find the button Pages and press
  • scroll down and find a option name branch
  • now set it main and save
  • reload that page and after some time maybe an hour, you will find your live link at the top.
